### JavaScript的诞生

> JavaScript的历史

​	JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在[HTML](https://link.zhihu.com/?target=https%3A//baike.baidu.com/item/HTML)（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。

> JavaScript诞生记

布兰登临危受命（1995年）

+ 公司要求给浏览器添加一个脚本功能
+ 公司要求脚本必须蹭Java的流量
+ 布莱登用十天时间设计JS的最初版本

>  JavaScript的10各设计缺陷

+ 不适合开发大型程序
+ 非常小的标准库
+ null和undefined
+ 全局变量难以控制
+ 自动插入行尾分号
+  加号运算符
+ NaN
+ 数组和对象的区分
+ == 和 ===
+ 基本类型的包装对象
